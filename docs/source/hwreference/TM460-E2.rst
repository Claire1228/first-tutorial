接口说明
--------

**网络接口**

TM460-E2 的网络接口采用 RJ45 接口。为确保网络连接稳定，请使用接头带固定螺丝的千兆网线。


**电源接口**

TM460-E2 电源接口型号为 HR10A-7R-6P，引脚定义如下图所示。

电源参数
----------

TM460-E2 采用外部直流供电。将外部直流电源，通过工业航插线缆连接到电源接口，即可为相机供电。推荐线缆型号为 HR10A-7P-6S。相机工作电源电压范围是直流 +12V~+24V。
